# programing-skill
=====

좋은코드란 도대체 어떠한 코드를 말하는것일까? 

> 좋은 코드의 정의

>> 1.유지보수성이 높다
>> 2. 신속하고도 효율적으로 동작한다
>> 3. 불필요한 부분이 없다.

> 습관 그 첫번째 - 읽기

>>  코드를 많이 읽어보는 것이 중요하다.

> 습관 그 두번째 - 쓰기
 
> 습관 그 세번째 - 두구의 연마

> 습관 그 네번째 - 알기 

> 숩관 그 다섯번째 - 듣기


>> 좋은 코드를 작성하기 위해서 필요한 일상의 습관에 대해 소개하였다. 자신의 형편에 맞춰 필요한 것부터 도입해 가면서 연습하는 것이 중요하다.
